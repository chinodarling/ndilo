❓ COMMON QUESTIONS
==================

Q: What if I get an API error when testing?
A: Check Netlify function deployment:
   1. Dashboard → Functions → Check "chat" is listed
   2. Check GROQ_API_KEY is in environment variables
   3. Redeploy site after adding env var
   See NETLIFY-FUNCTION-SETUP.txt for troubleshooting

Q: Chat still not working?
A: Check:
   1. Browser console (F12 → Console tab) for errors
   2. Netlify function logs (Dashboard → Functions → chat → Logs)
   3. Network tab - is /.netlify/functions/chat being called?
   Most common: API key not set in Netlify environment

Q: Can I change the WhatsApp number?
A: Yes! Find CONFIG.WHATSAPP_NUMBER = '27711285292' in HTML
   Replace with actual number (include country code)

Q: What if client upset about previous version?
A: Show her the fixes:
   1. Type different questions → Different responses
   2. Payment button works → Shows QR placeholder
   3. Tell her: "Just add your QR codes, I'll handle it"
