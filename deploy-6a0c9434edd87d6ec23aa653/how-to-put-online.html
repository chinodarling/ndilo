🐔 Put the Ndilo Chickens site online

No coding needed. This takes about 2 minutes.

Step 1 — Open the Netlify Drop page

Go to this website in your browser:

https://app.netlify.com/drop

You don't need to create an account first — just open that link.

Step 2 — Find the ndilo-chickens folder on your computer

Where is the folder?

You should have a folder called ndilo-chickens (the zip file was called ndilo-chickens.zip — unzip it first if you haven't already). Inside it you'll see files like index.html, netlify.toml, and an images folder.

Step 3 — Drag the folder onto the page

Click and drag the entire ndilo-chickens folder into the big grey "drag and drop" box on the Netlify page.

Wait a few seconds. A green bar will appear and a website address will be shown.

Step 4 — Done! 🎉

Your site is live!

Netlify gives you a free address like random-name-123.netlify.app. You can click it straight away to see the Ndilo Chickens website. Share that link with customers.

Optional: To get a custom name like ndilochickens.netlify.app, click Site settings → Change site name on Netlify.

⚠️ Important before you drag:
Make sure you drag the folder (called ndilo-chickens), not the zip file and not a file inside it.

If you only have the zip file: right-click it → Extract All (Windows) or double-click (Mac) → then drag the extracted folder.

If you want to add real photos later

Right now the site uses placeholder images. To add real photos, put your image files into the images folder and name them:

  • logo.png — your logo
  • hero.jpg — big background photo
  • product.jpg — chicken product photo
  • packaged.jpg — packaged chicken photo

Then drag the whole folder onto the Netlify drop page again to update the site.
