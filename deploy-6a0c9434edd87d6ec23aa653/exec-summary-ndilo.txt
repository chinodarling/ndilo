ARCHITECTURE - HOW IT WORKS

USER INTERACTION:
┌─────────────────────────────────────────────────────────────────┐

│ ↓                                                               │
│ callGroqAPI() called with full conversation history             │

│ ↓                                                               │
│ Frontend sends to: POST /.netlify/functions/chat                │
│ (no API key in request)                                         │
│ ↓                                                               │

│ ↓                                                               │
│ Function calls Groq API securely (backend-to-backend)           │
│ ↓                                                               │
│ Groq returns response: "5 chickens × R130 = R650.               │
│ 24 hours notice required. Collection or Ubersend?"              │
│ ↓                                                               │
│ Response sent back to frontend                                  │
│ ↓                                                               │
│ Typing indicator removed                                        │
│ ↓                                                               │
